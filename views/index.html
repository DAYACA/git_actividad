Programación

Según la página web Pixel Pro Cursos la programación es: "El proceso de escribir instrucciones y comandos para que una computadora realice tareas específicas. Se trata de comunicarse con la máquina a través de un lenguaje de programación para lograr un objetivo determinado. En esencia, la programación es la base de todo el software y aplicaciones que utilizamos en nuestros dispositivos electrónicos".

Existe una gran varieda de lenguajes de progrmaación, cada uno desarrollado para cumplir una tarea en especial, claro está que cada lenguaje puede ser usado para realizar más de una actividad para cual fue creado, en estos casos, se vería reducido de cierta forma ya que muchas de sus funcionalidades puede que no sean explotadas al máximo.

En esta web, hablaemos un poco más a fondo de estos leguajes:
